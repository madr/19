Varför har Windows så usla batteritider? →

2013-10-21 av Anders Ytterström

Jeff Atwood alias Coding Horror sammanställer obekväm sanning om Windows och batteritider. Han jämför med Mac OS X, som på samma hårdvara kramar ur 50-100% mer batteritid.

Microsoft prioriterar det inte, tycks det. Den sorgliga sanningen är att Linux hade presterat ännu sämre än Windows på samtliga maskiner som nämns i artikeln.
