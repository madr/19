Använd Keymaster för att enkelt fånga upp och delegera utifrån tangentbordstryck i webappar

2011-09-05 av Anders Ytterström

A simple micro-library for defining and dispatching keyboard shortcuts. It has no dependencies.

Från Keymaster på GitHub. Det här gillar jag. Dels för att det löser ett problem som alltid har funnits, men framförallt för att det inte kräver något tungt JavaScript-bibliotek som innehåller alldeles för mycket som inte används.

Nu saknar jag bara en javascript-baserad Front Controller i samma anda.
