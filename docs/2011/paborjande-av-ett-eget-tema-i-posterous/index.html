Jag vill såklart leva som jag lär och använda så mycket godis som möjligt. Utmaningen ligger i att skala av snarare än tillföra.

  • CSS Media-queries som elegant anpassar sajten Responsive Design istället för flera sajter med låsta dimensioner.
