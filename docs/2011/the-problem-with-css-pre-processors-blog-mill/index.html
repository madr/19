2011-11-21 av Anders Ytterström

This post is very polemic, I thought a lot before publishing it, please try to be open-minded and read the whole thing before commenting, it is not supposed to become a flame war. Please read the about page to understand the objective of this blog and the way I think about it.

*"With great power comes great responsibility."* -- Uncle Ben to Peter Parker

Det här är en bloggpost jag själv tänkt skriva många gånger men aldrig lyckats få ihop. Författaren har också erfarenhet av OOCSS, vilket jag inte ser som särskilt frapperande då OOCSS som filosofi avråder från vanor där pre-processors kan förvärra snarare än hjälpa.

Mycket, mycket glädjande läsning som rekommenderas för alla som nyss börjat med eller en längre tid använt Sass, Less eller Stylus.
