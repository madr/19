Introducerar madr.github.com, en blogg om min aktivitet på Github

2012-02-15 av Anders Ytterström

The markup is bootstraped by HTML5 Boilerplate and is generated by Jekyll. It looks pretty thanks to Skeleton. The font in the Top is called Ceasar Dressing, a free font from Google Web fonts.

All pages are valid HTML5. The CSS I wrote (less then 15 lines, I guess) are CSSLint free. No JavaScript was used for this site.

The site is hosted on GitHub Pages. No CMS, just Plain Old Semantic HTML and markdown.

via madr.github.com

Som så många andra webbutvecklare spenderar jag en hel del tid på Github, såväl i jobbet som personligen. Då jag ogärna vill börja kladda på engelska här på madr.se har jag gjort valet att skapa ännu en blogg.

Madr Code Rant kommer främst innehålla release notes för min egen kod och notiser som jag tycker är värt att nämna.

Som för att lära mig något nytt använde jag några verktyg för att bespara mig litet tid och krångel, som t ex Jekyll, Skeleton och HTML5 Boilerplate.

Det är en pirrande känsla att kunna blogga enbart med vim och git. Skoj och bizarrt på samma gång.
