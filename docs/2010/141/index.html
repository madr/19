Firecookie¶

Ger komplett kontroll över cookies från Firebug: listar, övervakar, ändrar och lägger till cookies bekvämt. Ett oumbärligt verktyg, då särskilt när aktuellt projektet har många komponenter som flitigt skriver cookies. Kan även sluta tillåta en eller flera domäner att skriva cookies.

Firefinder¶

Undrat hur många script en sida innehåller någon gång, och vilka som är inline och externa? Eller har det funnits behov av att snabbt räkna antalet element av något slag, t ex iframes? Då är det här verktyet värt en titt. Det söker i DOM-trädet utifrån ett uttryck i form av en CSS Selektor (CSS3 inkluderat) eller XPath. Resultat listas och hilightas och kan inspekteras direkt från listan.

Inline Code Finder¶

Besläktat med föregående, men i ett annat upplägg. Tillägget söker igenom DOM-trädet efter inline CSS och unobtrusive JavaScript och hilightar alla förekomster förutom att presentera i en överskådlig tabell. Ett gediget kvalitetstest på såväl annans kod som den egna.

Yslow¶

Yahoos numera trendsättande verktyg för att betygsätta prestandan för en sajt. Det finns ingen ursäkt för att inte ha den här installerad.

Google Page Speed¶

Googles motsvarighet till Yslow. Analyserar i stort sett detsamma, men har några egenskaper släktingen inte har. Kombon Page Speed + Yslow borde ses som tillräckligt för att bedömma en sajts prestanda.

CodeBurner¶

Sitepoints referens i ett smidigt, tillgänligt format. Bara högerklicka på ett element i inspektorn eller på sajten och få kompabilitetstabeller, urdrag ur W3C och exempel på användning. Gäller HTML-element och alla CSS-deklarationer.

Bubblare¶

Addons jag inte riktigt tagit ställning till ännu men som verkar lovande.

  • SenSEO - Någon form av sökmotorverktyg.
  • CSS Usage - undersöker hur mycket av nedladdad CSS som inte används på den aktuella sajten. Verkar lovande än så länge.
  • Validator - HTML-validator i Firebug. Jag gillar idén, men skulle nog föredra om CSS-validatorn också fanns.
  • Firediff - Borde vara av intresse för utvecklare till Ajaxsidor eller sidor med komplexa scripts.

Övergivna¶

  • Jiffy - rapporterar Jiffy-profilering till Firebugfönstret. Rätt användbart, men inte så relevant eftersom profilering i regel går ut på att jämföra resultat webbläsare mellan.
  • Pixel Perfect - Störde arbetet mer än främjade. Men en bra idé.
  • FireRainbow - Lyx, men när det gör Firebug långsamt går det att leva utan.
