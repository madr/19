Förvaltningsredovisning, route 1

Som jag sade igår när madr.se slängdes upp, så var inte allt riktigt testat och avslutat. Detta är det första av ett gäng förvaltningsmeddelanden som lär komma av och till.

Vad som åtgärdats:

Komihågfunktionen för bloggkommentarer : Fungerade inte pga ostädad kod. Åtgärdat.

Testning och avlusning i IE6 : Ginger-themet fungerar nu som jag vill i IE6. Har dock inte fått kika på sidan i IE7 ännu.

Allmänt fix : Några smådetaljer jag retade mig på. Nog sagt.

Bara positiva kommentarer än så länge. Det värmer. Många timmar ligger bakom den här ombyggnaden. :)
