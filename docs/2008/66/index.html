madr.se på Twitter

För de som inbillar sig att lösningar kring vertikal centrering med CSS och hantering av DOMNoder med ArrayAccess och Iterator blev skrivna på ett linjärt spår, rättframt och felfritt, kan jag upplysa att så inte är fallet.

Vägen kantas av svordomar, frustration, en hel del tankearbete med riklig mängd skit bakom spakarna. Sådant syns dock sällan i bloggar, eftersom bloggarna bara presenterar ett resultat.

För att dokumentera mina äventyr mer sanningtroget har jag skapat en Twitter där allt som rör webbutveckling och annars relevant till madr.se dokumenteras. För ökad sanningsenlighet ligger denna också som konversationsruta i min IM-klient.
